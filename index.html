postcss-typescale

Github

A PostCSS plugin to set type based on a typographic scale.

Input

@typescale {
  scale: 1.25;
  font-size: 1rem;
  line-height: 1.5rem;
}

.xxl {
  typescale: 3 2;
}

.xl {
  typescale: 2 1.5;
}

.l {
  typescale: 1 1.5;
}

.m {
  typescale: 0 1;
}

.s {
  typescale: -1 1;
}

.xs {
  typescale: -2 0.5;
}

Output

.xxl {
  font-size: 1.953125rem;
  line-height: 3rem;
}

.xl {
  font-size: 1.5625rem;
  line-height: 2.25rem;
}

.l {
  font-size: 1.25rem;
  line-height: 2.25rem;
}

.m {
  font-size: 1rem;
  line-height: 1.5rem;
}

.s {
  font-size: 0.8rem;
  line-height: 1.5rem;
}

.xs {
  font-size: 0.64rem;
  line-height: 0.75rem;
}
